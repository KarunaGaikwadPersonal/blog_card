# Frontend Mentor – Blog Preview Card Solution

This repository contains my solution to the Blog Preview Card challenge from Frontend Mentor. The project focuses on building a clean, responsive UI component using semantic HTML and modern CSS practices.

# 📌 Overview
## Objective

The goal of this challenge was to recreate a blog preview card component with attention to visual accuracy and user interaction. Key requirements included:

Replicating the provided design as closely as possible
Implementing hover and focus states for interactive elements
Maintaining clean structure and reusable styling

## 🛠️ Technologies Used
HTML5 (semantic structure)

# 📚 Key Learnings

This project reinforced several core front-end development concepts:

Writing semantic and accessible HTML for better structure and readability
Using Flexbox for layout alignment and spacing
Applying consistent design systems (spacing, typography, color usage)
Enhancing UI with subtle effects such as hover states and box shadows
